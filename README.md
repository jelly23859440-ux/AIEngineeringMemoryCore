# AIEngineeringMemoryCore
Time-based context solution.基于时间维度的上下文解决方案。
Project Introduction
I have absolutely no coding background. This tool was built entirely by describing my ideas to DeepSeek V4 Pro inside Trae. It is designed to address context memory challenges in medium and large-scale projects. I’m fully aware that the tool is far from perfect, but I believe this is a promising direction to make large language models more reliable — by mimicking human memory pathways and amplifying the strengths of AI. More details can be found in the accompanying markdown file.

As a non-programmer, I can currently only launch the program from the command prompt, and I haven’t been able to connect it with Trae to verify its actual effect or fix bugs. If an experienced developer could help optimize this tool and make it genuinely usable, I would be deeply grateful.
